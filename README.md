# Web 知乎日报
利用 React 搭建的一个在线 web 版知乎日报，用以解决知乎日报没有网页端又不想下载 app 的问题。
# 开发环境
+ react.js 16.2.0
+ es6
+ stylus
# 前言
首先得要有前端基础，其次加上一点 es6、stylus 就可以了。更多的一边写代码，一边查阅相关资料即可。这个项目是个比较简单的项目，我个人从 Vue 过来也懒得看教程，直接拿来即写，环境的搭建是参考了一些别人的资料，有些细节部分也不太清楚，在以后的开发中，力求搞懂所有的内容。
